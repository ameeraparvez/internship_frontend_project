# Modern Responsive Landing Page

This project was created as part of the **Web Development Internship – Task 1**.

The website is a modern and fully responsive landing page built using **HTML5, CSS3, and JavaScript**. It includes a professional UI design with smooth animations, glassmorphism effects, gradient backgrounds, feature cards, and a responsive layout for all devices.

---

## Features

- Responsive Navigation Bar
- Modern Hero Section
- Feature Cards with Icons
- Smooth Hover Effects & Animations
- Gradient & Glassmorphism UI
- Responsive Mobile Design
- Animated Statistics Section
- Call-to-Action Section
- Footer with Social Links

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Google Fonts
- Font Awesome Icons
